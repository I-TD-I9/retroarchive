# RetroArchive

Demo Node/Express backend with routes and a React frontend client

## Getting Started

1. Install dependencies for both the backend and frontend:
		- For the backend:
			```bash
			cd express-backend
			npm install
			```
		- For the frontend:
			```bash
			cd react-frontend-client
			npm install
			```

2. Start the backend server:
		```bash
		cd express-backend
		node server.js
		```

3. In a separate terminal, start the frontend development server:
		```bash
		cd react-frontend-client
		npm run dev
		```

6. Open your browser and navigate to `http://localhost:5173` to view the application.
